PROJECT NAME: TaskFlow Royal Suite
DEVELOPER: Anmol Srivastava
GITHUB REPO: https://github.com/AnmolSrivastava0323/TaskFlow-Anmol
TECHNOLOGIES: Node.js, Express, MongoDB, React (Frontend), Tailwind CSS

DESCRIPTION:
A premium project management application designed for professional teams. 
Features include project tracking, task management, team collaboration, 
and a high-end royal-themed dashboard.

HOW TO RUN LOCALLY:
1. Clone the repository.
2. Run 'npm install' to install dependencies.
3. Set your MONGODB_URI in a .env file.
4. Run 'npm start' to launch the server.
5. Access the app at http://localhost:3000.
